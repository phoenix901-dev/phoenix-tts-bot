# Phoenix TTS Bot

Асинхронный Telegram-бот для конвертации текста и книг в естественную речь. 

## Философия. Зачем это нужно?
Проект создан на принципах **цифрового суверенитета и честного железа**. 
Отказ от платных корпоративных API (Yandex, Google Cloud, AWS) в пользу прямых запросов к нейронным сетям Microsoft Edge. Вся тяжелая работа по парсингу, нарезке текста и рендерингу аудиопотока выполняется на вашем сервере. Вы получаете полный контроль над процессом генерации контента без лимитов на символы и скрытых подписок. Приоритет отдан физике потока и стабильности при обработке больших массивов данных (книг).

## Техническая база
* **Ядро:** Python 3.10+, Aiogram 3.x (Async)
* **Синтез:** `edge-tts` (Нейронные голоса без квот)
* **Медиа-движок:** `FFmpeg` (Конкатенация чанков в MP3 и автоматическая прошивка ID3-тегов)
* **Парсинг:** `Pandoc`, `pdftotext` (Извлечение текста с сохранением семантики), встроенный regex-чистильщик текста
* **БД:** `SQLAlchemy` + `aiosqlite`
* **Очередь:** Глобальная асинхронная очередь `asyncio.Semaphore` (защита от DDoS и перегрузки CPU)

## Что нужно поставить (Установка)

1. **Системные зависимости** (для работы с аудио и документами):
```bash
sudo apt update && sudo apt install ffmpeg pandoc poppler-utils -y
```

2. **Зависимости Python** (находясь в папке с ботом):
```bash
pip install -r requirements.txt
```

3. **Инициализация:**
Откройте файл `main.py` и пропишите токен вашего бота в переменную:
```python
API_TOKEN = "ВАШ_ТОКЕН"
```
*Убедитесь, что у скрипта есть права на запись в директорию `tmp`, указанную в `handlers.py`.*

4. **Запуск:**
```bash
python main.py
```

## Как пользоваться

После запуска бота и отправки команды `/start`:

1. **Мгновенный TTS (Голосовые сообщения):**
   Отправьте любой текстовый фрагмент. Бот асинхронно сгенерирует аудио и вернет его в формате нативного голосового сообщения Telegram (Opus, Voice).

2. **Создание аудиокниг из файлов:**
   Отправьте боту документ. Поддерживаемые форматы: `PDF`, `DOCX`, `FB2`, `EPUB`, `TXT`.
   * Бот самостоятельно извлечет текст с помощью `pandoc`/`pdftotext`.
   * Разобьет его на тома по ~200 000 символов (около 2.5 часов речи), чтобы обойти сетевые лимиты Telegram.
   * В 12 параллельных потоков отрендерит аудио.
   * Склеит фрагменты через FFmpeg и пришлет готовую многотомную аудиокнигу в формате MP3.

3. **Настройки параметров:**
   В меню `/settings` (или по кнопке "⚙️ Настройки") доступно управление:
   * **Выбор голоса:** Назначается раздельно для быстрых текстовых сообщений и для длинных книг (Доступны нейронные голоса RU/EN, мужские и женские).
   * **Скорость речи:** Регулировка темпа от -10% до +25%. Настройки сохраняются в локальной БД SQLite индивидуально для каждого пользователя.
